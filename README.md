s4a - Scratch for Arduino
-------------------------
[s4a](http://s4a.cat) is a scratch modification that allows for simple programming of the Arduino.


## Requirements


## Installation
arch linux
```sh
yay -S s4a
```

from source
```sh
git clone https://github.com/RaafatTurki/s4a
cd s4a
sudo make install
```

## Uninstallation
arch linux
```sh
yay -Rscn s4a
```

from source
```sh
sudo make uninstall
```
